API Basejump: Timestamp Microservice

FCC zipline by shshshjs1234@gmail.com

User Stories:
  1. I can pass a string as a parameter either in the url or directly in the input field, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a date or Unix timestamp, it returns null for those properties.

Try for example: January 1, 2016

{{tada}}

Example usage:

https://herokuapp.com/December%2015,%202015
https://herokuapp.com/252

Example output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
{ "unix": 252, "natural": "January 1st 1970, 1:04:12 am" }
